# FlowSync CRM Dashboard UI

A modern and scalable CRM Dashboard UI designed as part of my UI/UX Internship Task 3 at Future Intern.

## 📌 Project Overview

This project focuses on designing a professional B2B SaaS CRM dashboard used for managing:

* Leads
* Clients
* Tasks
* Follow-ups
* Team workflows
* Business analytics

The dashboard is designed with usability, scalability, and modern SaaS UX principles in mind.

---

# ✨ Features

## 📊 Dashboard Overview

* KPI summary cards
* Lead pipeline overview
* Recent activities
* Upcoming tasks

## 📋 Lead Management

* Lead tracking table
* Status-based pipeline
* Filters & sorting
* Search functionality
* Pagination support

## 👥 Client Details

* Client profile section
* Engagement overview
* Activity timeline
* Project progress tracking
* Contact & contract details

## ✅ Task Management

* Today's tasks
* Upcoming tasks
* Overdue reminders
* Completed tasks
